# davidpill47.github.io

Personal portfolio website built with Astro and Tailwind CSS.

## 🌐 Live

[davidpill47.github.io](https://davidpill47.github.io)

## 🛠️ Built with

- [Astro](https://astro.build) — static site framework
- [Tailwind CSS](https://tailwindcss.com) — styling
- [Formspree](https://formspree.io) — contact form
- [GitHub Pages](https://pages.github.com) — hosting

## ✨ Features

- Bilingual — Spanish and English
- Fully responsive — mobile and desktop
- Dark mode support
- Downloadable CV
- Contact form
- Deployed automatically via GitHub Actions

## 📁 Structure

src/
├── components/   # Hero, About, Skills, Experience, Projects, Education, Community, Contact
├── layouts/      # Main layout with Navbar
├── pages/        # ES (index.astro) and EN (en/index.astro)
├── styles/       # Global CSS and design tokens
└── i18n.ts       # All bilingual content

## 🚀 Run locally

```bash
npm install
npm run dev
```
